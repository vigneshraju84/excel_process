1.Install python 
2.Install virtual environment using following command
	pip install virtualenvwrapper-win
	
3.Create virtual environment
	mkvirtualenv django_demo
	
4.Than install django on virtualenv
	pip install django == 2.2.17
	
5.Open project folder in cmd than run following command for run then project
	python manage.py runserver

6.when the server is start successfully than open the url in browser 
	http://127.0.0.1:8000/

Process Flow
-------------
1.Choose excel file which contains Address column (sample excel file in "xl_files" folder in project)
2.Submit the form process will done and update the excel file with latitude and longitude values

Note:
	Choose file from "xl_files" folder from project folder
	Incase Choose file from another location than the updated file will store in "xl_files" folder
